# 💫 About Me:
I'm a backend systems engineer who builds things that runs the systems.I'm architecting real-time WebSocket pipelines serving 100+ concurrent clients, designing distributed event-driven systems with sub-100ms latency, and shipping production-grade code across the full stack.
I don't wait for someone to teach me. I pick up a problem, go deep, and deliver. Whether it's integrating NASA APIs, synchronizing collaborative editors across network failures, or building cross-platform monorepos with shared type-safe schemas — I figure it out and I build it right.
I'm not looking for a place to learn the basics. I'm looking for a place where the problems are hard enough to be worth solving.
